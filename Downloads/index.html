Source Code Release: Version 0.8.0 (September 7, 2002)

  • Download the latest official source code release.
  • Download the latest base package release.
  • Prerequisite: plib - portability libraries and scene graph. You need plib-1.6.0 or newer to build FlightGear 0.8.0.
  • Prerequisite: SimGear - simulation construction tools. You need version SimGear-0.2.0 of SimGear to build with FlightGear 0.8.0.
  • Documentation is shipped with the base package.

  • [Developers] Download a tarball of the latest Bleeding edge CVS snapshot.
  • [Developers] Download Previous versions.

Caution: Check the version numbers of the following prebuilt packages. There is usually a several day delay between the official source code release and when the ready to run binaries of the same version begin to show up here.

Ready to Run Windows Binaries

Note that windows binaries will be built sometime after the official source code release so expect a lag of a couple of days before the most recent version will show up here.
  • Download the fgfs-win32-bin-0.8.0 and fgfs-base-0.8.0 files here.
    Unzip these archives using something like pkzip or winzip into the same folder.
    Then go to the newly created FlightGear folder and run the "runfgfs.bat" file. The base package includes the same pdf manual that is listed at the top of this page. The manual is 109 pages and quite extensive. You are encouraged to read it first if you have any questions or problems. :-)
    The Win32 build is provided courtesy of Norman Vine.

Ready to Run Macintosh OS X (10.2) Binaries

Please note that at the moment these binaries require Mac OS 10.2 or newer.
  • Download the latest version here. You will also need the latest base package along with the executable.

    Please note that you will have to run this version from Terminal, for example using "./fgfs -fg-root=./fgfsbase."

Additional Scenery (Optional but Recommended)

You can download additional scenery if you like. The base package comes with a small scenery area so you can get up and running, but optionally you may download scenery for any portion of the world.
  • Download sections of the world: [ Graphical Interface ] [ FTP Interface ]. You can download scenery for much of the world in 10x10 degree chunks. The files are numbered by the coordinates of their lower, left hand corner. If you move your cursor over a tile on the image, your browser should display the name, size, and last modification date of the section.
