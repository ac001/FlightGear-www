Important Information

  • For a complete FGFS manual, please refer to the Installation and Getting Started Guide (PDF Version)
  • If you have trouble connecting to ftp.flightgear.org, please visit one of our FTP mirrors

Source Code Release: Version 0.9.8-pre2 (Updated January 3, 2005)

  • Download the latest source code release.
  • Download the latest base package release (71Mb). (Includes Textures, models, data, aircraft, sample scenery, and config files.) Alternatively: download a patch to upgrade the base package from version 0.9.6 to 0.9.8 (12Mb).
  • Prerequisite: OpenAL - Cross Platform 3D Audio. Strangely, OpenAL doesn't do versioned releases so you generally have to grab the latest CVS.
  • Prerequisite: plib - portability libraries and scene graph. You need plib-1.8.4 or newer to build FlightGear 0.9.8.
  • Prerequisite: SimGear - simulation construction tools. You need SimGear-0.3.8 to build FlightGear-0.9.8.
  • Documentation is included with the base package.

Developers
