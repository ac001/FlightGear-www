README! INSTRUCTIONS!

This form can be used to create aircraft configuration files for use with the JSBSim Flight Dynamics Model, the default FDM for the FlightGear flight simulator.

You will need at least two files for a complete configuration, an aero file containing information on the aircraft's aerodynamic properties, an engine file describing the engine(s), and in the case of a propeller-driven aircraft you will need a prop file. Aero-Matic will generate plausible configuration files for your aircraft using some simplifying assumptions. Note that Aero-Matic allows only one type of engine to be defined per aircraft. If you want to mix engine types you'll have to make the necessary changes by hand. Also note that all turbine, turboprop and rocket engines use the default "direct" thruster.

 This step is not necessary if you are using an already existing engine configuration file. In any case you will have to edit the propulsion section of the aero configuration file to ensure that the engine name is the same as the name of the engine configuration file.

You are ready to submit your data. Aero-Matic will send you a page called aero.php which contains your configuration file. Note that this file will probably look like garbage when viewed in your browser, but will look fine when viewing the document source. Save this file with a suitable filename, such as spitfireIX.xml. You will need to edit the propulsion section of this file so that the engine name is the same as the engine configuration file name, and the propeller name (if applicable) is the same as the propeller configuration file name.
