Important Information

  • If you have a slow/expensive net connection, like physical media, or want to help the FlightGear project, consider FlightGear on CDROM

  • For a complete FGFS manual, please refer to the Installation and Getting Started Guide (PDF Version)

  • If you have trouble connecting, please visit one of our FTP mirrors

  • Caution: Check the version numbers of the following prebuilt packages. There is usually a several day delay between the official source code release and when the ready to run binaries of the same version begin to show up here.

Ready to Run Windows Binaries (Updated for 0.9.4)

  • Download the self extracting/installing fgsetup-0.9.4.exe. (82 Mb)

  • After downloading the fgsetup-0.9.4.exe file, launch it and follow the on screen instructions to complete the installation. The installer will create start menu entries, install documentation, optionally install a desktop icon, and also will provide a well behaved uninstaller.

  • When you start FlightGear you will be presented with an initial "launch control" screen which allows you to select your aircraft, as well as a starting airport and runway.

  • FlightGear ships with an extensive 109 page manual. We encourage people to at least take a glance through it, especially if they have questions or problems.

  • The MSVC Win32 build is provided courtesy of Frederic Bouvier.

Ready to Run Sgi Irix Binaries (Updated for 0.9.4)

  • Download pre-built sgi binaries courtesy of Erik Hofman.

Linux Packages (Updated for 0.9.4)

  • Download pre-built Slackware packages courtesy of Jon Stockill.
  • Download pre-built Conectiva packages.
  • Gentoo ebuilds.
  • Download pre-built Red Hat Linux and Fedora packages courtesy of Richard Keech (rkeech 'at' redhat 'dot' com). If you are using Fedora, then you also need to install "glut" which can be found at http://rpm.livna.org/fedora/1/i386/RPMS.stable/glut-3.7-12.lvn.1.1.i386.rpm

Ready to Run Solaris Binaries (Updated for 0.9.4)

  • Download pre-built solaris binaries courtesy of Martin Spott.
  • The Solaris binary needs the GCC-3.3 runtime libraries which are included in the libgcc-3.3 package from Sunfreeware.com.
