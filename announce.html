January 18, 2005 - FlightGear v0.9.8 released.
Version 0.9.8 (source code) is available for download.
Download source code. (requires SimGear-0.3.8)

January 18, 2005 - SimGear v0.3.8 released.
Version 0.3.8 (source code) is available for download.
Download source code.

January 15, 2005 - Plib v1.8.4 is released.
FlightGear makes heave use of plib and you will need this new stable version to build the upcoming release of FlightGear. Here is the Plib home page.

December 19, 2004 - FlightGear is Kim Komando's Cool Site of the Day.
Kim Komando is "America's Digital Goddess". :-) FlightGear web traffic increased by about 10x over normal daily traffic on Saturday, Dec 18.

November 29, 2004 - FlightGear Interview Posted.
M station has posted an interview with Curtis Olson on the subject of FlightGear, scenery, flight dynamics, etc.

November 15, 2004 - French Language Web Site for FlightGear.
We now have a French translation of the FlightGear web site available thanks to the hard work of a Etienne Deparis: http://fr.flightgear.org/

November 1, 2004 - Japanese Language Web Site for FlightGear.
We now have a Japanese translation of the FlightGear web site available thanks to the hard work of a FG user named Tetsu: http://www.jp.flightgear.org/

October 15, 2004 - FlightGear IRC Channel.
Here is a reminder that we have an IRC channel available for live discussions:
Host = irc.flightgear.org, channel = #flightgear

October 14, 2004 - v0.9.6 for Windows released.
The Win32 build of v0.9.6 is now available for free download.
The Win32 v0.9.6 CDROM distribution is available for purchase (for those that want to save a large download or help support the FlightGear project.)

October 12, 2004 - FlightGear v0.9.6 released
Version 0.9.6 (in source code form) is available for download.
