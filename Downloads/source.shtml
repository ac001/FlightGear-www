Source Code Release: Version 0.9.8 (Updated January 18, 2005)

  • Download the latest source code release:
  • Download the latest base package release (71Mb). (Includes Textures, models, data, aircraft, sample scenery, and config files.)
    Or download a patch to upgrade the base package from version 0.9.6 to 0.9.8. (12Mb):
  • Prerequisite: Glut (FreeGlut) - OpenGL Utility Toolkit (GLUT) library.
  • Prerequisite: OpenAL - Cross Platform 3D Audio. Strangely, OpenAL doesn't do versioned releases so you generally have to grab the latest CVS.
  • Prerequisite: plib - portability libraries and scene graph. You need plib-1.8.4 or newer to build FlightGear 0.9.8.
  • Prerequisite: SimGear - simulation construction tools. You need SimGear-0.3.8 to build FlightGear-0.9.8.
  • Documentation is included with the base package.
