October 15, 2004 - FlightGear IRC

We now have two FlightGear IRC channels available. If you have time to burn, try one or both of them.
1. Host = "irc.linuxforen.de", channel = "#flightgear".
2. Host = "irc.flightgear.org", channel = "#flightgear".

October 14, 2004 - v0.9.6 for Windows

The Win32 build of v0.9.6 is now available for free download.
The Win32 v0.9.6 CDROM distribution is available for purchase (for those that want to save a large download or help support the FlightGear project.)

October 12, 2004 - FlightGear 0.9.6

Version 0.9.6 (in source code form) is available for download.
Download source code. (requires SimGear-0.3.7)
Download a ready to run version for your platform (as they become available.)

Dec 17, 2003 - Celebrate the Centennial of Flight with FlightGear

Here is a page with movies of the FlightGear 1903 Wright Flyer in action.

Oct 6, 2003 - Updated Scenery

The graphical scenery download interface has been updated to point to the latest scenery build.

Sep 30, 2003 - New website layout

Andrei Barbu has completely overhauled our website layout. There is still a few odds and ends to take care of. Please let someone know if you run into anything that looks like it got broke in the conversion.

Jan 29, 2003 - FlightGear receives "Best Free Simulation" Award

FlightGear wins the Linux Game Tome's award for the Best Free Simulation. Congratulations everyone!

Dec 20, 2002 - Wright Flyer model complete

Just in time for the start of the Centenial year of powered flight, FlightGear ships with a rather nice model of the 1903 Wright Flyer. The flight dynamics model was developed by Michael Selig at the University of Illinois, Urbana Champaign. The visual model started life as an MSFS aircraft, but was substantially reworked, enhanced, and animated by Jim Wilson of the FlightGear project.
