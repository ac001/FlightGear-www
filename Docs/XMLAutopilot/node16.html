Tweaking Parameters Inside of FlightGear

There is a slightly roundabout method for tweaking these parameters inside of FlightGear without needed to quit and restart.

  • Open the property browser and go to /autopilot/new-config/. There you will find an entry for each PID controller. They are numbered starting with zero and correspond to the order specified in your autopilot.xml config file.

  • Select the pid-controller that you would like to tweak. The entire set of parameters is available to be changed. The structure in the property browser exactly matches the structure in the config file.

  • Make any needed tweak. It's probably a good idea to only change one parameter at a time.

  • Close the property browser and select Reconfig Autopilot from the Autopilot menu.

  • Any parameter changes will take effect as soon as you select Reconfig Autopilot.

  • Changing the property tree does nothing to your autopilot.xml file so it's important to remember to edit your final tweaks into the config file before you exit FlightGear and forget what they are.
