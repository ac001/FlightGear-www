FGFS Tutorials

Additional tutorials are always welcome. FlightGear has a zillion features that aren't immediately obvious. Tutorials can be a great way to point out some of the more subtle features and details of FlightGear.
  • A basic flying tutorial using FlightGear.
  • A Circuit in FlightGear uses over 20 screenshots to show how to fly the FlightGear Cessna 172 smoothly by concentrating on attitudes.
  • FlightGear Flight School - A tutorial to walk you through the basics of flying an aircraft (set in the context of simulating all the procedures with FlightGear.)
  • Envoyez-vous en l'air ! - A FlightGear tutorial in French.
  • Building FlightGear with the free Cygwin compiler under Windows. Very detailed and well done. HTML Version. PDF Version.
