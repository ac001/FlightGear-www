Source Code Release: Version 0.9.3 (Updated October 24, 2003)

  • Download the latest official source code release.
  • Download the latest base package release. (Textures, models, data, and config files.)
  • Prerequisite: plib - portability libraries and scene graph. You need plib-1.6.0 or newer to build FlightGear 0.9.1.
  • Prerequisite: SimGear - simulation construction tools. You need version SimGear-0.3.4 of SimGear to build with FlightGear 0.9.3.
  • Documentation is shipped with the base package.

  • [Developers] Download a tarball of the latest Bleeding edge CVS snapshot.
