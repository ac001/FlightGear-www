Important Information

  • For a complete FGFS manual, please refer to the Installation and Getting Started Guide (PDF Version)
  • If you have trouble connecting to ftp.flightgear.org, please visit one of our FTP mirrors
  • Caution: Check the version numbers of the following prebuilt packages. There is usually a several day delay between the official source code release and when the ready to run binaries of the same version begin to show up here.

Ready to Run Windows Binaries (Updated for 0.9.2)

  • Download the fgfs-win32-msvc-bin-0.9.2.zip and fgfs-base-0.9.2.zip files.
  • Unzip these archives using something like pkzip or winzip. They should both be unzipped into the same place. A "FlightGear" folder will be created.
  • Go to the newly created FlightGear folder and run the "runfgfs.bat" file.
  • The base package includes the same pdf manual that is listed at the top of this page. The manual is 109 pages and quite extensive. You are encouraged to read it first if you have any questions or problems. :-)
  • The MSVC Win32 build is provided courtesy of Frederic Bouvier.
  • Norman Vine provides an alternate Win32 binary compiled with Mingw gcc. Note that this just hardwarereq.htms the MSVC binary package, you must also get the shared base package listed above.

Ready to Run Macintosh OS X (10.2) Binaries (Warning: version 0.8.0)

Please note that at the moment these binaries require Mac OS 10.2 or newer.
  • Download the latest version here. You will also need the latest base package along with the executable.

    Please note that you will have to run this version from Terminal, for example using "./fgfs -fg-root=./fgfsbase."

    The master/source location for the MacOS version is here.
